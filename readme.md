# Hexo 博客管理客户端

## 功能

*   [ ] 初始化 hexo 项目（如果使用的话，自动配置为 next 主题） 界面上打印终端日志
*   [ ] 对 hexo 以及主题进行 form 形式的配置(毕竟修改配置文件的方法对很多人来说还是太 geek 了，一期只打算做 next 主题)
*   [x] 管理 hexo 中的的 post 和 draft 中的文件，辅助进行 markdown 的写作
*   [x] 利用命令行进行草稿和文章的新建
*   [x] 实时预览 md 文件
*   [x] 一键执行`hexo g -d` 实时打印终端日志，进行远程部署
*   [x] 插入图片自动上传七牛并替换链接
